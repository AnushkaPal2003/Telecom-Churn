# 📊 Telecom Customer Churn Prediction

This project predicts whether a telecom customer will churn using machine learning models.

---

## 🚀 Project Overview

Customer churn is a major problem in the telecom industry. This project builds a machine learning system to identify customers likely to leave the service so that retention strategies can be applied.

---

## 🧠 Models Used

- Logistic Regression (Best Model)
- Decision Tree
- Random Forest
- K-Nearest Neighbors
- XGBoost
- Gaussian Naive Bayes
- Support Vector Machine
- Voting Classifier

---

## 📊 Evaluation Metrics

- Accuracy
- F1 Score
- ROC-AUC
- Recall (important for churn detection)

---

## 🏆 Final Model

**Logistic Regression**

Reason:
- Best ROC-AUC score
- Balanced performance
- Easy to interpret
- Good business usability

---

## ⚙️ Project Workflow

1. Data Cleaning

4. Model Training
